Adatkezelés XML-ben

Tantárgy tematikus leírása:

XML adatmodell. XML dokumentum elemei. A DTD szabvány. Az XMLSchema szabvány elemei. Adatelemek hivatkozása. xPath szabvány. Integritási szabályok definiálásának lehetőségei. Az XML kezelése gazdanyelvi környezetben, a SAX és DOM API szabványok áttekintése és programozása Java környezetben. Dokumentum konverzió : XSL és XSLT. Az XSLT lehetőségei: feldolgozás menete, szelekció, ciklusképzés, feltételes végrehajtás, csoportképzés, aggregáció, változók használata, dinamikus struktúra felépítés. XML adatok generálása adatbázisokból. SQL/XML szabvány elemei. JSON modell elemei. JSON és XML összevetése, JSON séma kezelése. JSON API programozása.
